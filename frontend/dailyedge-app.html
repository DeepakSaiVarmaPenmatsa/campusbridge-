Two Sum
Easy +50 XP
Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

You may assume that each input would have exactly one solution, and you may not use the same element twice. You can return the answer in any order.
Constraints
• 2 ≤ nums.length ≤ 10⁴
• -10⁹ ≤ nums[i] ≤ 10⁹
• -10⁹ ≤ target ≤ 10⁹
• Only one valid answer exists
Example 1
Input: nums = [2, 7, 11, 15], target = 9
Output: [0, 1]
Explanation: nums[0] + nums[1] == 9
Example 2
Input: nums = [3, 2, 4], target = 6
Output: [1, 2]
Test Cases
Output
▷ nums = [2,7,11,15], target = 9 → Expected: [0,1] Not Run
▷ nums = [3,2,4], target = 6 → Expected: [1,2] Not Run
▷ nums = [3,3], target = 6 → Expected: [0,1] Not Run

💡 AI Hint
Think about using a hash map to store values you've seen. For each number, check if its complement (target - num) already exists in your map. This gives you O(n) time complexity instead of the brute force O(n²).

Question 1 of 1 · Logical Series
⏱️ 02:00
What is the next number in the series?
2, 6, 12, 20, 30, ?
A
38
B
40
C
42
D
44
Explanation
The series follows the pattern n(n+1): 1×2=2, 2×3=6, 3×4=12, 4×5=20, 5×6=30, 6×7=42

Differences: 4, 6, 8, 10, 12 → increasing by 2 each time. The next difference is 12, so 30 + 12 = 42 ✓

Perspicacious
/ˌpɜːr·spɪˈkeɪ·ʃəs/
adjective
Having a ready insight into things; shrewd and having a clear understanding of complex situations. Often used to describe someone with sharp mental acuity.
The perspicacious interviewer immediately noticed inconsistencies in the candidate's resume, asking precise follow-up questions that revealed the truth.
How is "perspicacious" best used in a professional context?
A
The perspicacious weather was sunny and warm all day long.
B
Her perspicacious analysis of the market trends helped the team make a profitable investment.
C
He perspicaciously ate his lunch in the conference room.
D
The perspicacious building had 42 floors and was recently constructed.
Why B is correct
"Perspicacious" means having keen insight or shrewd understanding. Option B correctly applies it to analytical ability — the quality of her analysis. Options A and D misapply it to inanimate things, and C uses it as an adverb incorrectly.
